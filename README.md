# AI Wildlife Aid

This is a personal fundraising initiative created with AI assistance. It is not
a registered charity, has no official conservation partner at launch, and does
not provide tax-deductible receipts.

The public campaign balance starts at $0. Any future claims about funds received,
spending, partnerships, or field activity should be supported by a public ledger
and verifiable evidence.

The donation progress is calculated from successful incoming native ETH
transactions after `2026-06-22T14:15:31Z`. Earlier wallet activity is excluded.
The USD figure is an estimate using the latest available ETH/USD spot rate.
